# Day 22 – Model Deployment with Flask

## Overview
Today's focus is deploying a trained ML model as a REST API using Flask.

## Topics Covered
- Saving and loading models with `joblib`
- Building a Flask API endpoint
- Sending POST requests with JSON payloads
- Testing the API locally

## Project
Wrapped the AI Sales Analyzer prediction model in a Flask app with a `/predict` endpoint that accepts input features and returns a sales forecast.

## Key Learnings
- Flask routing and request handling
- Model serialization best practices
- REST API design for ML models
- CORS handling for frontend integration

## Tools & Libraries
- Flask
- joblib
- scikit-learn
- Postman (for testing)
